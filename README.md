# EduFarm Website

Official website for EduFarm, a non-profit organization dedicated to educating, empowering, and sustaining communities through innovative agricultural practices, youth development, and environmental sustainability.

## Project Goals

- Promote sustainable agriculture
- Empower youth through education
- Support community development
- Showcase EduFarm's programs and impact

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
